Changes

  • When the browser shows a search result page from one of the installed search engines, the awesomebar displays the search terms instead of the URL of the page (regardless of how you've reached that page).
  • The URL of the search result page is available via a contextual menu or by pressing ESC when the cursor is in the awesomebar.
  • The icon on the left of the awesomebar changes according to how the query will be interpreted: a URL shows the globe, a search of any kind shows the magnifying glass
  • This is not a new mode. As soon as you start typing/editing inside the awesomebar, the usual suggestions appear.
